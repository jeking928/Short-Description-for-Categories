PURPOSE & AIM

Basically this allows you to SPLIT your category descriptions into two parts. One part comes before any products or subcategories (as a normal zen cart install does), and another part comes AFTER any products or subcategories. This mod also places a "more info..." link directly after the first description, so your customers can "jump" down to the secondary description.

This add on will be useful for people who have LENGTHY category descriptions, and who don't wish to force their customers to scroll down to find the products or subcategories.

If there is NO secondary category description, there will be no "more info..." link. You are also able to adjust the "more info..." text via a language file, and you can remove it all together if you'd prefer not to have that link.

Ready to get started? Just follow the steps in the installation tab

INSTALLATION

!!!!!BACKUP YOUR STORE BEFORE YOU BEGIN!!!!!

This version is specifically for Zen Cart 1.5.4 If you are using an older version, you MUST download and use the apprporiate older version.

New Installation

  1. Unzip the package to your local hard drive.
  2. Rename the YOUR_TEMPLATE and YOUR_ADMIN folders to match the name of your shop's template and admin directories.
  3. Use the included sql file to update your database (adds a new row to the categories_description table). Copy and paste the contents into the text box at Tools>Install SQL Patches. Then click 'send'.
  4. There are two core overrides for this mod, as they have no override folder. If you have already modified these files, you will need to merge the changes to these files here into your version of these files. Make a backup of your current files before editing and replacing them. The two core overrides for this plugin are:
    • admin/categories.php
    • includes/modules/pages/index/main_template_vars.php
  5. Upload the files to your website, retaining the folder structure.
  6. DONE! Now when you go to add a category or edit a category in your admin, there will be a new section where you can enter the description that goes below the category or product listings.
  7. (OPTIONAL) Add this to your stylesheet.css (color and padding for the "more info...." text)
    • #descSubLink {
      padding-left:.7em;
      color:#990000;
      }
  8. (OPTIONAL) You can adjust the "more info..." text in includes/languages/english/extra_definitions/YOUR_TEMPLATE/category_description_sub_defines.php

Un-Installation

Reverse everything you did. Remove all the files and undo the edits from the modified core files. Better yet, replace with the backups you created before installing this plugin. It is best to simply keep a good pre-install backup set so you can restore if you decide this isn't what you want.

USAGE


Very simple. If you want to split a category descritiption, go to the category in the admin and fill in the 'Categories Description' with the text you want above the products/sub-categories and fill in the 'Categories Description Below Categories' box with the text you want below the products/sub-categories.
